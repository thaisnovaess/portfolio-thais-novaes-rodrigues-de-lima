🎯 Proposta de Valor

🚀 Instruções de Uso

Para o Treinador

1.
Abra o FitCall no seu dispositivo Android

2.
Clique em "Criar Treino"

3.
Insira o nome da sala (ex: "Treino Força - Segunda")

4.
Insira seu nome (ex: "Personal João")

5.
Clique em "Iniciar Treino"

6.
Compartilhe o código com o aluno

7.
Use as ferramentas durante a sessão:

📊 Cronômetro: Acompanha tempo total

⏱️ Timer: Intervalo de 30 segundos

🔢 Contador: Registre repetições



Para o Aluno

1.
Abra o FitCall no seu dispositivo Android

2.
Clique em "Entrar em Treino"

3.
Insira o código fornecido pelo treinador

4.
Insira seu nome (ex: "Maria")

5.
Clique em "Entrar"

6.
Acompanhe o treino em tempo real com o treinador

Permissões Necessárias

Na primeira execução, o FitCall solicita:

✅ Câmera: Para videochamada
✅ Microfone: Para áudio
✅ Internet: Para conectar ao Jitsi

Autorize todas as permissões para funcionamento completo.

📋 Instalação Passo a Passo

Requisitos
Android Studio Iguana ou superior
JDK 17+
Dispositivo Android API 24+ ou emulador

Passos:

Bash

# 1. Clone o repositório
git clone https://github.com/seu-usuario/fitcall-android.git
cd FitCall

# 2. Abra no Android Studio
# File → Open → Selecione a pasta FitCall

# 3. Aguarde a sincronização do Gradle
# Android Studio sincronizará automaticamente

# 4. Compile o APK
./gradlew assembleDebug

# 5. Instale no dispositivo
adb install app/build/outputs/apk/debug/app-debug.apk

# 6. Ou execute direto do Android Studio
# Run → Run 'app'


🔗 Integração Jitsi Meet

O FitCall utiliza o Jitsi Meet SDK para videochamadas. A integração é automática e não requer configuração adicional:

✅ Servidor público: meet.jit.si
✅ Sem necessidade de servidor próprio
✅ Criptografia ponta a ponta
✅ Qualidade adaptativa
✅ Suporte a múltiplos participantes

Como funciona: Quando você cria uma sala no FitCall, o app gera um código único que conecta a uma sala Jitsi correspondente. Alunos usam esse código para entrar na mesma sala.

Erro: "Gradle sync failed"

Bash


./gradlew clean
./gradlew build



Erro: "Permission denied for camera"

Verifique permissões em Settings → Apps → FitCall → Permissions

Erro: "Jitsi SDK not found"

Certifique-se de que os repositórios Maven estão configurados em settings.gradle
